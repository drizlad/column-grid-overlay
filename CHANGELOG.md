# Changelog

All notable changes to the Column Grid Overlay extension.

## [1.0.4] - 2026-02-26

### Fixed
- Grid not appearing when toggle is clicked
- "Could not establish connection" error on some pages

### Added
- Programmatic content script injection
- Friendly message on restricted pages

## [1.0.3] - 2026-02-26

### Fixed
- Simplified content script message handling

## [1.0.2] - 2026-01-28

### Fixed
- Connection errors on restricted pages
- Grid persisting across all tabs

### Changed
- Grid visibility is now per-tab

## [1.0.1] - 2026-01-28

### Fixed
- "Failed to load config" error
- "Failed to toggle grid" error
- Content script initialization issues

## [1.0.0] - 2026-01-28

### Added
- Initial release
- Customizable column grid overlay (1-24 columns)
- Adjustable gutters and margins
- Color picker with opacity control
- Built-in presets (Bootstrap, Tailwind, Material Design)
- Custom preset save/load
- Keyboard shortcut (Ctrl+Shift+G / Cmd+Shift+G)
- Settings persistence
- Manifest V3 compliance
